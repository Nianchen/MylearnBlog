# 引言

我们日常学习的过程中，笔记之间穿插图片都是很常见的一个现象，但是通过markdown记录的图片在另一个电脑中就会因为缓存的问题无法显示，这时候我们就需要把图片上传到云端进行保存。

# 所需工具

1.Typora:markdown文档编辑器 2.PicGo图片上传器 3.github账号

# 具体流程

1.下载并安装Typora以及PicGo

2.打开Typora点击偏好设置 Ctrl + 逗号

找到图像-->插入图片时-->选择上传图片 之后配置好自己安装的PicGo路径

这时候就已经可以上传照片了但是我们还没有对PicGo进行配置

3.PicGo配置

我们找到PicGo的图床设置，因为图床有很多，我们这里选择配置Github图床

这里我们就需要申请一个Github账号来进行下一步操作了，具体申请过程就不在描述了

# Github配置

1.申请过后，我们在自己的账号中新建一个public的仓库，也就是开放的仓库，比如我这里的LearnImg仓库

2.最后生成的路径就是：你的用户名/你创建的仓库名

3.分支一般默认的都是主分支也就是：mian

4.生成token，点击自己头像后依次找到-->setting-->developersettings-->Personl access tokens-->tokens生成一个token也就是身份验证

5.在picgo中填入这个token

6.这个自定义域名是因为github是外网需要cdn加速如果不加速的话可能上传很慢
